### 1. What is a Palindrome?
- A palindrome is a string that reads the same forward and backward.

Example:
```js
Forward  : racecar
Backward : racecar
```

### 2. What is the Problem?
**Given a string s, find the longest substring that is a palindrome.**
***Important:***
- Substring must be continuous.

Example:
```js
Input: s = "babad"

Possible palindromes:
bab
aba

Output: "bab" or "aba"

// Another examples
Input: "cbbd"

Substrings:
c
b
b
d
bb

Output: "bb"
```
